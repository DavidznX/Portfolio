Projeto de Aprendizagem em Desenvolvimento de Jogos


Introdução

Este projeto é criado apenas para fins de aprendizagem individual. Decidi implementar várias mecânicas em um único jogo.


Sistema de Armas

  • Pegar arma
  • Atirar
  • Recarregar

Sistemas Básicos Gerais

  • Vida
  • Dano
  • Ataque
  • Movimentação
  • Outros

Sistema de Veículos

  • Entrar em carros
  • Sair
  • Dirigir
  • Movimentação de veículos

Sistema de Combate com Armas Brancas

  • Espadas
  • Lanças
  • Outras

Sistema de Habilidades e Poderes Especiais

  • Kamehameha
  • Rasengan
  • Lapso
  • Reversão
  • Vazio Roxo

Objetivo do Jogo

O jogo consiste em explorar o mapa, construir suas builds e utilizar os recursos e habilidades à disposição para derrotar inimigos que surgem para te atacar.
